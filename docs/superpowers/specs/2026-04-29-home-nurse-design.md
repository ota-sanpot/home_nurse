## 1. コンセプト

訪問看護への転職を考える看護師（病院からの転職・ブランク明け）向けのSEO記事メインのアフィリエイトサイト。
Google検索からの自然流入を主な集客チャネルとし、訪問看護ステーションの求人案件へのCVで収益化する。

**ターゲット読者**
- 病院勤務から訪問看護に転職を検討している看護師
- ブランクがあり、復職先として訪問看護を考えている看護師

**マネタイズ**
- 訪問看護・看護師転職求人のアフィリエイト案件（ASPは今後選定）

## 2. 技術スタック

| 項目 | 選定 | 理由 |
|------|------|------|
| フレームワーク | Next.js 14（App Router） | SEO・パフォーマンス・静的エクスポート対応 |
| 記事管理 | MDX | Markdownで記事を量産しやすい |
| スタイリング | Tailwind CSS | 開発速度とカスタマイズ性の両立 |
| ホスティング | Vercel（無料プラン） | Next.jsとの相性が最良 |
| デプロイ | `next export`（静的HTML生成） | 高速・低コスト |

### 各ページ詳細

#### トップページ `/`
- **ヒーローセクション**: キャッチコピー＋サブテキスト＋CTAボタン（求人サイトへ誘導）
- **注目記事グリッド**: 最新・人気記事を2×2〜2×3で表示
- **カテゴリナビ**: 主要カテゴリへのリンク

#### 記事一覧 `/articles`
- 全記事をカード形式で一覧表示
- タイトル・概要・カテゴリ・投稿日を表示

#### 記事詳細 `/articles/[slug]`
- **レイアウト**: 1カラム（スマホ最適化）
- **構成**:
  1. 記事タイトル・メタ情報
  2. リード文＋「この記事でわかること」ボックス
  3. 本文（MDX）
  4. **本文途中にアフィリCTAバナーを挿入**（自然な位置）
  5. **記事末尾にアフィリCTAバナー**（大きめ）
- MDX frontmatterでtitle・description・category・dateを管理

#### 運営者情報 `/about`
- 運営者のプロフィール・サイトの目的を記載
- YMYL（健康・医療・お金）カテゴリのSEO信頼性向上に必須

#### プライバシーポリシー `/privacy`
- アフィリエイトサイト運営に必須の免責・プライバシー記載

## 5. SEO設計

### メタタグ自動生成
MDX frontmatterの`title`・`description`からNext.jsの`generateMetadata`でOGP・メタタグを自動生成。

### 記事カテゴリ
| カテゴリ | 想定キーワード例 |
|----------|-----------------|
| 病院からの転職 | 「訪問看護 転職 病院 違い」「訪問看護 転職 後悔」 |
| ブランク明け | 「看護師 ブランク 復職 訪問看護」「ブランク 訪問看護 未経験」 |
| 給与・待遇 | 「訪問看護 給与 相場」「訪問看護 年収 上がる」 |
| 転職サイト比較 | 「訪問看護 転職サイト おすすめ」「看護師 転職 ランキング」 |

### 内部リンク戦略
- 記事内から関連記事へリンクを張り、回遊率を高める
- トップページの注目記事グリッドで主要記事への導線を確保

## 6. コンポーネント設計

### `AffiliateCTA`
アフィリエイトリンクを差し込むCTAコンポーネント。記事内に複数回呼び出せる設計。

```tsx
// 使用例（MDX内）
<AffiliateCTA
  title="訪問看護の求人を無料で見てみる"
  href="https://affiliate-link.example.com"
  buttonText="求人を見る →"
/>
```

### MDX frontmatter 形式
```mdx
---
title: "訪問看護への転職を成功させる5つのポイント"
description: "病院から訪問看護に転職する方法を経験者目線で解説します。"
category: "病院からの転職"
date: "2026-04-29"
---
```

## 7. 開発・デプロイフロー

1. `npx create-next-app@latest` でプロジェクト初期化
2. Tailwind CSS・MDX・`@next/mdx` のセットアップ
3. コンポーネント・ページ実装
4. `next.config.ts` に `output: 'export'` を設定
5. Vercelにリポジトリ接続・自動デプロイ設定
6. 記事MDXファイルを`content/articles/`に追加するだけで更新可能
