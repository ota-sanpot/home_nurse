給与と同じくらい重要なのが福利厚生です。しかし訪問看護の求人では福利厚生の記載が少なく、実態がわかりにくいことがあります。本記事で詳しく解説します。

## 社会保険（健康保険・厚生年金）

常勤（週30時間以上）で働く場合、社会保険（健康保険・厚生年金保険・雇用保険・労災保険）の加入は法律で義務付けられています。

| 勤務形態 | 社会保険 |
|------|------|
| 常勤（週30時間以上） | 加入義務あり |
| パート（週20〜30時間） | 条件を満たせば加入可 |
| パート（週20時間未満） | 国保・国民年金で自己対応 |

求人票に「社会保険完備」と記載されている場合は、4つの保険（健康保険・厚生年金・雇用・労災）がすべて揃っています。

## 退職金制度

訪問看護の退職金制度は、運営主体によって大きく異なります。

| 運営主体 | 退職金の傾向 |
|------|------|
| 医療法人・社会福祉法人 | 退職金制度あり（中退共・独自制度） |
| 大手株式会社 | 退職金制度あり（運営会社による） |
| 中小株式会社・個人経営 | 退職金なしのケースが多い |

「中小企業退職金共済（中退共）」に加入している事業所は、在籍年数に応じた退職金が保証されます。

## 研修・スキルアップ支援

訪問看護では、スキルアップ支援が充実している事業所が増えています。

**研修支援の種類：**

| 種類 | 内容 |
|------|------|
| 外部研修費用補助 | セミナー・学会参加費を補助 |
| 資格取得支援 | 認定看護師・ケアマネ取得費用の補助 |
| 院内勉強会 | 月1〜2回、スタッフ全員で参加 |
| eラーニング | オンライン研修の受講支援 |

認定看護師取得の費用（数十万円）を法人が負担してくれる事業所もあります。

## 住宅手当・家賃補助

住宅手当や家賃補助を設けている事業所もあります。

- **住宅手当**：月5,000〜30,000円が一般的
- **借り上げ社宅**：会社が契約した物件に格安で入居できる制度

特に都市部の大手法人系ステーションでは、家賃補助が充実しているケースがあります。年収が同程度でも、住宅手当があることで実質的な生活水準が大きく変わります。

## その他の福利厚生

| 制度 | 内容 |
|------|------|
| 健康診断 | 年1回（法定）＋各種オプション検査補助 |
| インフルエンザ予防接種 | 費用補助または無料接種 |
| 保育施設の利用補助 | 院内保育・提携保育所・費用補助 |
| 財形貯蓄 | 給与から天引きして貯蓄 |
| 福祉施設の優待 | 提携ジム・リゾート施設の割引 |

## 福利厚生の確認方法

求人票だけでは全容がわかりにくいため、面接・見学時に積極的に確認しましょう。

**面接で聞くべき質問：**
- 「退職金制度はありますか？何年以上在籍で支給対象になりますか？」
- 「外部研修や資格取得への費用補助はありますか？」
- 「住宅手当や家賃補助はありますか？」
- 「育休・産休を取得したスタッフはいますか？」

## まとめ

訪問看護の福利厚生は、医療法人・大手法人の運営では充実していることが多い一方、小規模・個人経営では最低限にとどまるケースがあります。給与だけでなく退職金・研修支援・住宅手当を含めた「トータルパッケージ」で比較することが、後悔しない転職のポイントです。
