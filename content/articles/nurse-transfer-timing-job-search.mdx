「転職したいけど、いつ動き出せばいいの？」というお悩みを解決します。看護師転職のベストタイミングと、スケジュールの組み方を解説します。

## 看護師求人が増える時期

看護師の求人数は時期によって変動します。特に求人が増えるのは以下の時期です。

### 1〜3月（最多）

年度末に向けて退職者が増えるため、4月の人員補充を見越した求人が大量に出ます。最も求人数が多い時期です。

### 9〜10月

秋の人事異動・年度中途採用に向けて求人が増えます。1〜3月ほどではありませんが、良い求人が出やすい時期です。

### 比較的求人が少ない時期

- 6〜8月：夏季は求人の動きが鈍くなりがち
- 11〜12月：年末で人事が動きにくい

## 内定から入職までの期間

訪問看護の転職における一般的なスケジュールは以下の通りです。

| フェーズ | 目安期間 |
|------|------|
| 転職サイト登録〜求人紹介 | 1〜2週間 |
| 応募〜面接 | 1〜2週間 |
| 面接〜内定 | 1〜2週間 |
| 内定〜入職 | 1〜2か月 |
| **合計（転職活動開始から入職まで）** | **2〜4か月** |

つまり、4月入職を希望するなら1月には転職活動を開始する必要があります。

## 現職の退職手続きとの兼ね合い

多くの病院・事業所では、退職を**1〜3か月前**に申し出ることが就業規則で定められています。

**スケジュール例（4月入職の場合）：**

| 時期 | アクション |
|------|------|
| 1月 | 転職サイト登録・求人探し開始 |
| 1〜2月 | 面接・内定 |
| 2月 | 現職への退職申し出（退職日：3月末） |
| 4月 | 新しい職場に入職 |

退職申し出を遅らせると「引き留め」「後任問題」で入職が遅れることがあります。内定が出たら速やかに現職に申し出ましょう。

## 急いで転職したい場合の動き方

「今すぐ転職したい」「3か月以内に転職したい」という場合は、以下のポイントを意識しましょう。

- **即日〜翌日対応の転職サイトに複数登録する**
- **「急募」「即日入職可」の求人を優先して探す**
- **面接を複数社同時に進める**
- **現職の有給を活用して退職時期を前倒す**

急いでいる場合でも、職場の雰囲気確認（見学）はできる範囲でしておくことをおすすめします。

## 転職を急ぐべきでないケース

逆に、焦って転職しない方が良いケースもあります。

- **賞与を受け取ってから退職したい場合**：支給日の直前に退職すると賞与をもらえないことがあります
- **有給が多く残っている場合**：退職前に消化できるか確認する
- **資格取得が間近の場合**：資格取得後に転職する方が給与交渉で有利

## まとめ

看護師の転職活動は「入職希望日の2〜4か月前」に始めるのが理想です。求人が増える1〜3月・9〜10月を狙いつつ、現職の退職手続きと連動させてスケジュールを組みましょう。転職サイトのアドバイザーに「〇月に入職したい」と伝えると、逆算してサポートしてもらえます。
