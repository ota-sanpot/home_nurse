訪問看護師として働き続けながら「もっと収入を上げたい」と考えることは自然なことです。本記事では、現実的に年収を上げるための5つの方法を具体的に解説します。

## 訪問看護師の平均年収から考える

まず現状を整理します。訪問看護師の平均年収は450〜550万円程度と言われています。

| 経験年数 | 目安年収 |
|------|------|
| 1〜3年 | 350〜420万円 |
| 3〜7年 | 420〜500万円 |
| 7〜10年 | 500〜560万円 |
| 管理職 | 550〜700万円 |

これを踏まえた上で、年収アップのための具体策を見ていきましょう。

## 方法1：オンコール手当を積極的に活用する

最も手軽に月収を上げる方法がオンコール手当の活用です。

- 待機手当：1回3,000〜5,000円
- 出動手当：1回5,000〜20,000円

月8〜10回の待機をこなすと、毎月3〜5万円の上乗せが期待できます。年換算で36〜60万円のアップになります。

**注意点：** 出動頻度が高い事業所はその分疲弊しやすいため、待機回数と出動率のバランスを事前に確認することが大切です。

## 方法2：資格を取得してスキルアップ

資格手当がつく資格を取得すると、基本給のベースが上がります。

| 資格 | 年収アップの目安 |
|------|------|
| 認定看護師 | 30〜50万円/年 |
| 専門看護師 | 50〜80万円/年 |
| ケアマネジャー | 20〜40万円/年 |
| 特定行為研修修了 | 30〜60万円/年 |

特定行為研修は近年注目を集めており、「特定看護師」として医師の包括指示のもとで診療補助行為を行える資格です。訪問看護との相性が非常に高く、取得者は引く手あまたになっています。

## 方法3：好条件の事業所へ転職する

同じ訪問看護師でも、事業所によって給与は大きく異なります。

**給与が高い事業所の特徴：**
- 医療法人・大手グループ運営
- 24時間対応で手当が充実している
- 専門特化型（小児・精神科・難病など）

**転職時の交渉ポイント：**
- 現在の年収を伝えた上で希望年収を提示する
- 経験・スキルを具体的にアピールする
- 複数社の内定をもとに交渉する（転職サイト活用が効果的）

## 方法4：管理者・管理職を目指す

管理職になることで、給与は大幅にアップします。

| ポジション | 年収目安 |
|------|------|
| 主任・リーダー | 450〜520万円 |
| 管理者・ステーション長 | 550〜700万円 |
| 複数ステーション管理 | 650〜900万円 |

管理者になるには「専従の看護師」であることが要件となっており、基本的に常勤正職員であることが前提です。管理者候補として採用してもらえるかを面接時に確認するのも一つの方法です。

## 方法5：副業・ダブルワークを活用する

訪問看護ではパート・非常勤として複数の事業所で働く「ダブルワーク」が可能です。

本業の訪問看護に加えて：
- 週末のみ別の訪問看護で働く
- デイサービスや有料老人ホームでの非常勤勤務
- 単発・スポット勤務（看護師派遣）

こうした掛け持ちにより、年収を50〜100万円上乗せしている看護師も珍しくありません。

## まとめ

訪問看護師の年収を上げる方法は複数あります。

1. **オンコール手当の活用** → 今すぐできる
2. **資格取得** → 中長期で確実に上がる
3. **高条件事業所への転職** → 交渉次第で大幅アップ
4. **管理職を目指す** → キャリアアップとセットで
5. **副業・ダブルワーク** → 短期間での収入増に有効

まずは自分の現状を把握し、どの方法が一番現実的かを検討してみましょう。転職サイトに相談すると、現在の市場価値を教えてもらえることもあります。
