訪問看護は「ワークライフバランスが取りやすい」と言われますが、実際のところどうなのか気になる方も多いはず。本記事では、訪問看護師の実際の働き方をリアルに解説します。

## 訪問看護師の1日のスケジュール例

| 時間 | 業務内容 |
|------|------|
| 8:30 | 出勤・朝の申し送り・その日の訪問スケジュール確認 |
| 9:00 | 1件目の訪問出発 |
| 9:00〜10:00 | 1件目訪問（バイタル測定・点滴管理・記録） |
| 10:10〜11:10 | 2件目訪問 |
| 11:20〜12:10 | 3件目訪問 |
| 12:10〜13:00 | 昼休憩（車内や事務所） |
| 13:00〜14:00 | 4件目訪問 |
| 14:10〜15:00 | 5件目訪問 |
| 15:10〜16:10 | 6件目訪問 |
| 16:30 | 事務所に戻り記録・報告書作成 |
| 17:30 | 退勤 |

1日5〜7件の訪問が標準的です。訪問と訪問の間の移動時間は10〜20分程度です。

## 残業の実態

訪問看護の残業は、病棟と比べると少ないのが一般的です。

**残業が発生する主なケース：**
- 急変・緊急訪問が入った場合
- 記録・報告書の作成が追いつかない
- スタッフ会議・研修が定時後にある

**残業が少ない理由：**
- 訪問スケジュールは事前に組まれており、突発的な業務が少ない
- 夜勤がなく、日中の業務が明確に区切られている
- 電子カルテ・スマートフォンで記録を現場で完結させやすい

多くの訪問看護師は月の残業が10〜20時間以内に収まっていると報告しています。病棟での残業時間（月30〜60時間以上という声も）と比べると、大幅に少ない傾向があります。

## プライベートの確保しやすさ

### 定時退勤ができる

夜勤がなく、定時近くに退勤できることが多いため、家族との夕食・子どもの寝かしつけ・趣味の時間を確保しやすくなります。

### 自分のペースで業務が進められる

訪問の合間の移動時間は自分のペースで過ごせます。車移動の場合は好きな音楽・ラジオを聴きながら移動する看護師も多いです。

### 服装が自由

訪問看護の制服はナース服ではなく、動きやすいポロシャツ・スクラブが多いため、着替えの手間が少ないのもメリットのひとつです。

## ワークライフバランスが取りにくいケース

一方で、以下のような状況ではバランスが崩れやすいです。

- **オンコールの頻度が高い事業所**：月15回以上の待機は負担が大きい
- **利用者の急変が多い時期**：ターミナル期の利用者が複数重なると緊張状態が続く
- **スタッフが少ない事業所**：1人当たりの件数が多く、記録が追いつかない

## ワークライフバランスが良い職場の見極め方

| 確認項目 | 理想の回答 |
|------|------|
| 月平均残業時間 | 20時間以内 |
| オンコール待機回数 | 月6回以下 |
| 1日の訪問件数 | 5〜7件 |
| スタッフの定着率 | 高い（3年以上のスタッフが多い） |
| 育休・産休取得実績 | 取得者がいる |

## まとめ

訪問看護はワークライフバランスが取りやすい職場が多いですが、事業所によって差があります。「残業時間」「オンコール頻度」「スタッフ定着率」を転職前にしっかり確認することで、自分らしい働き方を実現できる職場を選びましょう。
