「訪問看護に転職したいけど、どんな流れで進んでいくの？」という方のために、応募から入職までの全ステップを詳しく解説します。

## 全体の流れ

```
転職サイト登録 → 担当者との面談 → 求人紹介 → 応募 → 書類選考
→ 面接（1〜2回） → 内定 → 条件確認・承諾 → 現職への退職申し出
→ 入職前準備 → 入職
```

## ステップ1：転職サイトへの登録（1〜2日）

転職サイトに登録すると、担当アドバイザーから連絡が来ます（電話またはメール）。

**最初のヒアリングで伝えること：**
- 希望勤務地・勤務形態（常勤/パート）
- 希望年収
- オンコールへの対応可否
- 転職希望時期
- ブランクの有無

## ステップ2：求人紹介（1〜2週間）

担当アドバイザーから条件に合った求人が紹介されます。複数の求人を比較し、気になるものを選びます。

**求人を選ぶ際の確認ポイント：**
- 月給・賞与・手当の詳細
- 勤務時間・週の勤務日数
- オンコールの頻度と手当
- 訪問エリアの範囲
- スタッフ数と利用者数

## ステップ3：応募・書類提出（3〜5日）

担当アドバイザーが代わりに応募手続きをしてくれます。

**必要な書類：**
- 履歴書（手書き・PC作成どちらも可）
- 職務経歴書（要求される場合）
- 看護師免許証のコピー

書類に不安がある場合は、アドバイザーに添削を依頼しましょう。

## ステップ4：面接（書類通過後1〜2週間）

訪問看護の面接は通常1〜2回です。

| 面接回数 | 内容 |
|------|------|
| 1次面接 | 担当者・管理者との面談。経歴・動機・条件確認が中心 |
| 2次面接（ある場合） | 理事長・院長など上位者との面談 |

**面接でよく聞かれること：**
- 転職理由・訪問看護を選んだ理由
- オンコールへの対応可否
- 一人訪問への不安はあるか
- ターミナルケアへの考え方

**逆質問の例：**
- 「同行訪問期間はどのくらいですか？」
- 「オンコールは月何回程度ありますか？」
- 「スタッフの平均勤続年数はどのくらいですか？」

## ステップ5：内定・条件確認（面接後1週間以内）

内定の連絡は、面接後3〜7日以内が多いです。

**内定後に必ず確認すること：**
- 給与の詳細（基本給・各種手当の内訳）
- 賞与の金額・支給時期
- 試用期間の有無と期間・試用期間中の給与
- 入職日・雇用形態
- 雇用契約書の発行タイミング

曖昧な条件は後のトラブルになります。「雇用契約書で確認させてください」と言うのは当然の権利です。

## ステップ6：現職への退職申し出

内定を承諾したら、できる限り早く現職に退職の意思を伝えます。

**退職申し出のポイント：**
- 就業規則の「退職申し出期限」を確認する（多くは1〜2か月前）
- 入職日の1〜2か月前を目安に申し出る
- まず直属の上司に口頭で伝え、その後退職届を提出する

## ステップ7：入職前準備

入職までの間に準備しておくこと：

- 健康診断書（事業所から指定があれば）
- 制服・ユニフォームのサイズ確認
- 訪問に使う文房具・携帯電話の準備
- 基本的な処置の手順確認（吸引・点滴管理など）

## まとめ

訪問看護の転職は、登録から入職まで2〜4か月が目安です。各ステップで確認することを怠らず、焦らず進めることが後悔しない転職の第一歩です。転職サイトを活用すれば、書類・面接・交渉のすべてをサポートしてもらえます。
